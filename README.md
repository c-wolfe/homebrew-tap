# homebrew-tap

[HomeBrew](http://brew.sh/) thirdparty repository ([TAP](https://docs.brew.sh/Taps)).

## Vert.x 4 formula

Usage:

```
> brew tap vertx-distrib/tap
> brew install vertx4
```

After you can use Vert.x CLI:

```
> vertx -version
```

## Vert.x 3 formula

Usage:

```
> brew tap vertx-distrib/tap
> brew install vertx3
```

After you can use Vert.x CLI:

```
> vertx -version
```
